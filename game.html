Random Number Game

The rules of the game are:

  • When the game starts you have 1000 INR
  • Every time you press Start, 3 random numbers are genrated and 100 INR gets deducted
  • If all the random numbers are either odd or even, you get 300 INR
  • If all the random numbers are in sequence (eg 123,012, ... etc), you get 800 INR
  • If all the random numbers are same,you get 1000 INR
  • If INR reaches 0, Game Over
